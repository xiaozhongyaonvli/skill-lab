# skills-lab

分享我创建的实用 AI Skills，覆盖多种任务场景与工作流方向。

## 仓库说明

这个仓库用于集中管理我自己创建、整理和迭代的 Skills。  
每个 Skill 独立放在自己的目录中，便于后续单独维护、测试和复用。

后续新增 Skill 时，只需要：

1. 在仓库根目录新增对应的 Skill 文件夹
2. 在下面的「Skill 列表」中追加一项
3. 如有必要，同步更新该 Skill 的使用说明

## 目录结构

```text
skills-lab/
├─ README.md
├─ interest-to-pr-finder/
└─ ...
```

## Skill 列表

### 1. interest-to-pr-finder

路径：`interest-to-pr-finder/`

用途：
用于根据用户感兴趣的方向，筛选适合贡献的 GitHub 仓库与 issue。

当前能力：

- 根据主题做仓库检索与候选收敛
- 支持基于编号的仓库选择
- 将选择结果和筛选过程落盘
- 支持 issue 候选结果持久化
- 支持全局偏好与缓存目录

适用场景：

- 想找适合自己参与的开源仓库
- 想根据技术方向筛选 issue
- 想把仓库筛选与 issue 调研过程沉淀下来，便于后续继续推进
